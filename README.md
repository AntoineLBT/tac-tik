# Tac-tik
Projet de tac-tik en c++ avec la plus bonne bonne de mes copines, Laulau !
C'est moi :)

## get project
```
mkdir projet
cd projet
git clone https://github.com/Extro96/tac-tik.git
```

## Build project
```
mkdir build
cd build
cmake ..
make
```

## launch project
```
cd build
./Tactik
```
